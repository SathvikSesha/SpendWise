# SpendWise
<p>The application is a full-stack, user-centric budget management platform built using the React frontend, Node.js with Express.js backend, and MongoDB as the database. It enables users to create and manage multiple independent budgeting “spaces” tailored to specific financial contexts such as events, travel, or personal planning. Each space is initialized with a defined budget limit, optional threshold for alerts, and descriptive metadata. Within a space, users can dynamically log expenses by categorizing them under user-defined, case-insensitive categories, allowing flexible and real-world expense structuring without predefined constraints. The system continuously tracks total expenditure and computes the remaining budget in real time, while maintaining category-wise aggregation for clear financial visibility. A threshold-based alert mechanism is implemented to notify users via email when their spending approaches a predefined percentage of the total budget, ensuring proactive financial control without restricting further spending. The architecture follows a scalable, normalized data model by separating spaces, categories, and expenses into distinct collections, enabling efficient updates, extensibility, and future optimizations such as caching and analytics. Authentication is handled via both OAuth and traditional email-based login enhanced with multi-factor authentication using OTP verification, ensuring secure access control. The platform is designed with a focus on modularity, data consistency, and real-world usability, with future scope for advanced analytics, visualization, and collaborative budgeting features.</p>
